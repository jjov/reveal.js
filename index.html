Synthetic monitoring by store

APPROACHES

  • To use store id http header in PAM Proxy.
  • To add synthetic header in PAM when expected store is in the canonical request.

To use store http header in PAM Proxy.

  • Http header present in most of carrier calls in PAM. (X-PAM-EXTERNAL-STOREID)
  • Some carrier requests have to be modified in PAM.
  • Two properties in PAM Proxy, to know if the deployment is prod, and to know which stores will be for synthetic monitoring.
  • To add the logic for synthetic monitoring by store in PAM Proxy.

To add synthetic header in PAM

  • Add a new property to know which stores are for synthetic monitoring.
  • To implement the logic in ApiServiceEndPointBase class from integrationcomponents.
  • To add synthetic monitoring header when the store is the expected, and the deployment is prod.
