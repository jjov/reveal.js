GetDeviceType In RetrieveAccount

  • As part of Tracfone Auto-Refill implementation was necessary to remove getDeviceType from Create Customer canonical workflow and add it to Retrieve Account canonical API call to validate if the phone number which is going to recharged belongs to an active account.
